Commands TO Run After Installing Visual Studio and Node In Visual Studio Code Terminal:

npx create-expo-app SheSafety   ---> To create a app with "SheSafety" name.

cd SheSafety

Download EXPO GO app in your mobile and create a account.

Run this command in terminal:

npx expo login --> Login With Account Created In Your Mobile.

npx expo start  --> You Will Get A QR Scan with the app and you can see you app running.

Note Points: Connect with same wifi in same device.Its better to use mobile hostpot for pc while scanning and viewing.
